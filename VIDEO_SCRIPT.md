# 🎥 2-Minute Video Demo Script

## Setup Before Recording

- [ ] Open `/demo` page in browser
- [ ] Clear browser console
- [ ] Set browser zoom to 100%
- [ ] Close unnecessary tabs
- [ ] Test audio/video quality
- [ ] Have backup recording ready

---

## Script (2 minutes)

### Opening (0:00 - 0:15)

**[Show demo page]**

> "Hi! I'm excited to show you CareerPilot AI - a live AI agent that applies to jobs using Gemini 2.0 Flash Vision."

**[Point to hero section]**

> "Unlike traditional chatbots, our agent streams its thoughts in real-time and shows you exactly how confident it is about each action."

---

### The Problem (0:15 - 0:30)

**[Scroll to show the interface]**

> "Job applications are time-consuming and repetitive. You fill out the same information over and over."

**[Point to the form inputs]**

> "Our agent solves this by visually understanding job application pages - no APIs, no DOM access - just pure computer vision with Gemini."

---

### Live Demo (0:30 - 1:30)

**[Click "Start Agent" button]**

> "Let me show you. I'll click Start Agent..."

**[Watch the live feed]**

> "Watch this - the agent is now analyzing a real LinkedIn job posting for a Software Engineer role at Stadium."

**[Point to confidence score as it updates]**

> "See the confidence score? 92%. The agent isn't guessing - it's showing us exactly how confident it is based on visual clarity, label matching, and data availability."

**[Point to live activity feed]**

> "And here's the magic - you can see its thoughts in real-time. It's not just executing actions, it's explaining its reasoning with personality."

**[Point to progress bars]**

> "The agent is now filling in the application form - first name, last name, email, phone, location - all automatically."

**[Wait for success]**

> "And... success! Application submitted. You can see the confirmation screenshot right here."

---

### Technical Highlights (1:30 - 1:50)

**[Show the metrics cards]**

> "Let's talk about what makes this special:"

**[Point to each card]**

> "First, live streaming with Server-Sent Events - not turn-based chat."

> "Second, confidence scoring with detailed reasoning - proving the AI isn't hallucinating."

> "Third, visual UI understanding using Gemini 2.0 Flash - works on any job board."

**[Scroll down to supported platforms]**

> "And it works universally - LinkedIn, Indeed, Glassdoor, any career portal."

---

### Google Cloud Integration (1:50 - 2:00)

**[Show deployment info or mention it]**

> "Everything runs on Google Cloud Run with Cloud Build for CI/CD, and we use Secret Manager for secure API key storage."

> "It's production-ready and scales automatically."

---

### Closing (2:00 - 2:10)

**[Back to top of demo page]**

> "So that's CareerPilot AI - a live AI agent that breaks the text-box paradigm with streaming, confidence scoring, and visual understanding."

**[Smile at camera]**

> "Thanks for watching! Try the demo yourself at the link below."

## Recording Tips

### Do's ✅
- Speak clearly and enthusiastically
- Point to UI elements as you mention them
- Let the demo run without interruption
- Show genuine excitement
- Keep energy high throughout

### Don'ts ❌
- Don't rush through the demo
- Don't apologize for anything
- Don't say "um" or "uh" too much
- Don't look away from camera too long
- Don't forget to smile!

---

## Technical Setup

### Screen Recording
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 30 fps minimum
- **Audio:** Clear microphone, no background noise
- **Software:** OBS Studio, Loom, or QuickTime

### Browser Setup
- **Zoom:** 100%
- **Window Size:** Maximized
- **Extensions:** Disable ad blockers
- **Theme:** Light mode (better for recording)

### Lighting
- Face well-lit
- No harsh shadows
- Natural or soft lighting

---

## Backup Plan

If live demo fails:
1. Have pre-recorded demo ready
2. Explain it's a backup recording
3. Continue with confidence
4. Mention it works live (show later if possible)
